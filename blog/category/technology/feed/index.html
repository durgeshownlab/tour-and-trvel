The best way to demonstrate horizontal image with your blog post. This theme allows you to add images with flexible height so that you can express what you want.

The Cocept Behind

The overall concept is very simple, You just need an image with width greater than 1140px and of any height. When you upload the image as featured image from WordPress post editor, WordPress automatically crop the images to 1140px width and proportional height of no matter what value. This way if you upload an image that has less height and more width then you end up having a horizontal image as featured image.
